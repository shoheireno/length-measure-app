# 写真で長さ測定（PWA版）

写真の中に「長さがわかっている基準物」（定規、名刺、硬貨など）を一緒に写し、
2点をタップして基準を設定すると、以降は写真上の好きな2点間の実寸をピクセル比から計算して表示するWebアプリです。

インストール不要・Apple Developer登録不要。SafariでURLを開くだけで使え、
「ホーム画面に追加」すればアイコンからアプリのように起動できます。

## 使い方（アプリ内の流れ）

1. 「写真を撮る」または「ギャラリーから選ぶ」で対象の写真を読み込む
2. 写真内の、実際の長さがわかっている2点をタップ（例: 定規の0cmと10cmの目盛り）
3. ダイアログでその実際の長さと単位（mm/cm/m/inch）を入力 → 基準が確定
4. 以降、測りたい2点をタップするたびに、実寸が計算されて線とラベルが表示される
5. 右上の ↺（1つ戻す）/ ⟳（全リセット）、各測定行の✕で編集可能

## iPhoneで使う方法

このフォルダをどこかのWebサーバーに置き、iPhoneのSafariでそのURLを開きます。2通りの方法があります。

### 方法A: 同じWi-Fi内でPCから配信（すぐ試せる）

1. Windows PCでこのフォルダ内の `serve.ps1` を実行してローカルサーバーを起動
   ```powershell
   cd C:\Users\inada\Desktop\AI\length_measure_web
   powershell -ExecutionPolicy Bypass -File serve.ps1
   ```
2. PCのLAN内IPアドレスを確認
   ```powershell
   ipconfig
   ```
   「IPv4 アドレス」（例: `192.168.1.23`）を確認
3. iPhoneを**同じWi-Fi**に接続し、Safariで `http://192.168.1.23:8787/` を開く
4. 共有ボタン（□に↑）→「ホーム画面に追加」でアイコン化

   ※ HTTP（非HTTPS）なので、オフラインキャッシュ（Service Worker）は効きませんが、
   カメラ撮影・測定機能は問題なく使えます。PCとiPhoneが同じWi-Fiに繋がっている間だけ使えます。

### 方法B: 無料の静的ホスティングに公開（どこからでも使える・推奨）

GitHub Pages / Netlify / Cloudflare Pages などの無料サービスに `length_measure_web` フォルダ一式を
アップロードすると、自動でHTTPSのURLが発行されます。HTTPSなのでオフラインキャッシュも有効になり、
ホーム画面のアイコンから常時アプリのように使えるようになります。

（デプロイ作業は、無料ホスティングサービスのアカウント作成が必要なため、進める場合は改めて手順をご案内します）

## 精度に関する注意

- 基準物と測定対象は「同じ平面上」かつ「カメラに対してできるだけ正対」して撮影してください。奥行きや傾きがあると誤差が大きくなります。
- 基準物はできるだけ大きく（＝多くのピクセル数で）写すほど誤差が減ります。
- 広角レンズの端など、歪みが大きい部分に対象が写っていると誤差が出やすいので、画面中央付近に収めて撮影するのがおすすめです。

## コード構成

```
index.html   画面構造（写真選択ボタン、キャンバス、校正ダイアログ）
style.css    スタイル
app.js       ロジック（画像読込・タップ処理・校正・描画・測定リスト）
manifest.json / icon.svg   PWA設定（ホーム画面アイコン用）
sw.js        オフラインキャッシュ用Service Worker（HTTPS/localhostでのみ有効）
serve.ps1    ローカル動作確認用の簡易HTTPサーバー（PowerShell）
```
